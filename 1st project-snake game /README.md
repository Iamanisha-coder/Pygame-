# 🐍 Pygame Snake Arcade

A classic graphical Snake game built with Python and the **Pygame** library. This project features smooth grid-based movement, real-time score tracking, and collision detection.

---

## 🚀 How to Run

### 1. Prerequisites
Make sure you have Python installed. You will also need the `pygame` library.

### 2. Installation
Clone the repository and install the dependencies:
```bash
git clone [https://github.com/Iamanisha-coder/1st project-snake game.git]
cd snake-pygame
pip install -r requirements.txt
